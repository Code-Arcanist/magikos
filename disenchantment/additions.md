# Additions

Added in a game rule "magikos-disenchantment:allow_overpowered_enchantments" which will turn off:
* Maximum Enchantment Levels in the Codex Combiner
* Allow multiple conflicting enchantments on compatible items, e.g. Protection and Blast Protection on the same chestplate.
